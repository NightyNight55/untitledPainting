# 緣起

用於展示2024暑假Revival Camp 劇本殺使用的正反面線索卡，
設計主架構的我們覺得花了那麼多時間設計的東西，就這麼讓它消失了也有點可惜，
所以就以這樣的方式讓他繼續保存著吧！
也算是給前段時間一直爆肝想這些東西的我們一個交代吧～
